Inject custom jQuery/Javascript into the header and/or footer of a WordPress site.

== Description ==
Inject-O-Matic is a WordPress plugin that allows users to inject custom jQuery and Javascript into the header and footer of a WordPress site.

== Installation ==

1. Old-school: upload the `inject-o-matic` folder to the `/wp-content/plug-ins/` directory via FTP.  Hipster: Ironically add Inject-O-Matic via the WordPress Plug-ins menu.
1. Activate the Plug-in
1. Test that the this plug-in meets your demanding needs.
1. Rate the plug-in and verify if it works at wordpress.org.
1. Leave a comment regarding bugs, feature request, cocktail recipes at http://wordpress.org/support/plugin/inject-o-matic
